# Nutriplan – Recipe & Nutrition Tracker

## Overview
Nutriplan is a web-based application designed to help users manage recipes and track nutritional intake. The app provides detailed nutrition data, including calories and macronutrients, to support healthier decision-making.

## Features
* Search and view recipes with detailed nutrition information
* Display macronutrient breakdown (protein, carbs, fats)


## Technologies Used
* JavaScript
* HTML/CSS
* Spoonacular API
